# Fix It Please !! — v1.5 Release Notes

---

## 🇺🇸 English (en-US)

**What's new in v1.5:**

• Multi-room repair requests — select multiple locations and issues in a single request
• Blunt, detailed request writing — generated descriptions are direct, specific, and avoid legal language
• Multi-photo support — attach several photos to one repair request
• Communication timeline — log responses and follow-ups for each request
• Document Vault improvements — organize files by category (lease, photos, receipts, correspondence)
• Voice notes — record notes by speaking or typing
• Export full repair history as a shareable document
• Performance and stability improvements

## 🇺🇸 Spanish — United States (es-US)

**Novedades en v1.5:**

• Solicitudes de reparación con múltiples cuartos — selecciona varias ubicaciones y problemas en una sola solicitud
• Redacción directa y detallada de solicitudes — las descripciones generadas son claras, específicas y sin lenguaje legal
• Soporte para múltiples fotos — adjunta varias fotos a una solicitud de reparación
• Cronología de comunicación — registra respuestas y seguimientos para cada solicitud
• Mejoras en el Almacén de Documentos — organiza archivos por categoría (contrato, fotos, recibos, correspondencia)
• Notas de voz — graba notas hablando o escribiendo
• Exporta tu historial completo de reparaciones como un documento compartible
• Mejoras de rendimiento y estabilidad
